## Prerequisites

Before running the scripts, ensure you have **Python 3.14** or higher installed on your system.

*   **Windows / macOS / Linux:** Download from [python.org](https://www.python.org/downloads/).

## Installation

1.  Clone this repository or download the ZIP file.
2.  Open your terminal (Command Prompt or Terminal) and navigate to the folder.
3.  Install the required dependencies (e.g.,SymPy) using pip:

    ```bash
    pip install -r requirements.txt
    ```

## Usage

To verify a specific field equation, run the corresponding script. For example, to verify the geometric constraints:

```bash
python 01_tt_component_constraint.py
